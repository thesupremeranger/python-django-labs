Лабораторные работы по предмету «Прикладные информационные системы».

## Лабораторные работы

**Lab1 — Основы Python. Установка Django.**
Написание первых скриптов на Python: списки, словари, функции. Установка Django, создание проекта, знакомство с административным интерфейсом.

**Lab2 — Первая веб-страница.**
Создание простой страницы через `HttpResponse`. Подключение HTML-шаблона через `render`. Настройка статических файлов: CSS и изображения.

**Lab3 — Модель данных.**
Создание модели `Article` с полями и методами. Регистрация модели в админке. Вывод всех записей из базы данных через динамический шаблон с циклом.

**Lab4 — Страница отдельной записи.**
Маршруты с параметрами через регулярные выражения. Страница конкретной статьи по её `id`. Верстка обеих страниц по макету.

**Lab5 — Форма создания поста.**
HTML-форма с полями названия и текста. Обработка GET и POST запросов. Валидация данных, сохранение в базу и редирект.

**Lab6 — Регистрация и авторизация.**
Создание пользователей через `django.contrib.auth`. Формы регистрации и входа. Проверка уникальности никнейма и корректности пароля.

**Lab7 — Основы JavaScript.**
Базовый синтаксис JS: переменные, функции, циклы. Подключение скриптов к шаблону. Сворачивание и разворачивание постов по клику через работу с DOM.

**Lab8 — jQuery.**
Подключение библиотеки jQuery. Подсветка поста при наведении через `.hover()` и `.animate()`. Анимация логотипа при наведении.

**Lab9 — Эффект параллакса.**
Обновление дизайна главной страницы. Три иконки разного размера в шапке. Реализация параллакса через отслеживание прокрутки `$(window).scroll()`.
